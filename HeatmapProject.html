Study Spot Heat Map Project Plan

A project aimed at making a system of wireless packet sniffers to keep track of the number of clients in a particular location -- in ths case, a designated ‘study spot’ -- on a college campus, therefore allowing us to estimate the relative activity of that location. Using this data, we can create a visualization to help students decide where their ideal study location would be at any given time.

Requirements

Primary

Scanning Module (single location):

  • Ability to count MAC addresses of nearby clients
  • Ability to read signal strength of nearby clients
  • Ability to ‘score’ clients weighted by their signal strength
  • Ability to report aggregate score of all nearby clients to server

Server:

  • Database of all locations and their scores
  • Ability to display current location activity data graphically via CSS / HTML / JS / AJAX

Visualization Station (Milner):

  • Ability to display current location activity data.

Secondary

Scanning Module (single location):

  • Ability to distinguish between device manufacturers based on OUI
  • Ability to triangulate location of clients
  • Any other statistical research we can thinkof

Physical display (tenuous):

  • Ability to read data from server and control a large number of LEDs individually based on that data
  • Weather proof
  • Tamper proof resistant
